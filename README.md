# Zomato Restaurant Data Analysis

## Project Overview
This project analyzes Zomato restaurant data to uncover insights about restaurant ratings, pricing, online delivery trends, and location-based performance. The objective is to understand key factors that influence restaurant success on Zomato.

## Dataset

The project uses the original Zomato restaurant dataset stored in the raw data directory.

*Dataset Location:*
data/raw/zomato_raw.xlsx

*Description:*
The dataset contains detailed information about restaurants listed on Zomato, including:

- Restaurant name and city
- Cuisines offered
- Average cost for two
- Online delivery availability
- Ratings and votes

The raw dataset was cleaned and transformed before analysis.
## Data Pipeline

The project follows a structured data analysis pipeline:

1. Raw data is stored in data/raw/
2. Data cleaning and preprocessing is performed in Excel
3. Cleaned datasets are stored in data/processed/
4. Analysis and dashboards are generated from processed data

This structure ensures reproducibility and professional data management practices.
## Tools & Technologies

## Objectives

- Analyze distribution of restaurant ratings
- Identify top cities with highest number of restaurants
- Study relationship between cost and ratings
- Compare restaurants offering online delivery vs offline only
- Discover most popular cuisines

## Tools & Technologies

- Microsoft Excel
  - Data Cleaning
  - Pivot Tables
  - Charts & Dashboards

## Key Analysis Performed

- Data cleaning and removal of duplicates
- Handling missing values
- City-wise restaurant analysis
- Rating vs Cost analysis
- Online delivery impact on ratings
- Cuisine popularity analysis

## Key Insights

- Major metro cities dominate restaurant listings
- Restaurants with online delivery tend to receive higher ratings
- Moderate pricing range restaurants attract more customers
- Certain cuisines consistently receive higher ratings

## Conclusion

This analysis provides useful insights for restaurant owners and food-tech companies to improve customer experience, pricing strategies, and service availability.
